# 9D Cửu Long Tranh Bá - Game Clone

Game 3D RPG võ hiệp lấy cảm hứng từ 9D, xây dựng trên Godot Engine 4.

## Yêu cầu

- [Godot Engine 4.2+](https://godotengine.org/download/)

## Cách chơi

1. Tải và cài đặt Godot 4.2+
2. Mở Godot, chọn "Import" và trỏ đến thư mục `9d-game/`
3. Nhấn "Run" (F5)

## Điều khiển

| Phím | Hành động |
|------|-----------|
| WASD / Mũi tên | Di chuyển |
| Chuột trái | Chọn mục tiêu / Tấn công |
| Chuột phải kéo | Xoay camera |
| 1-4 | Sử dụng kỹ năng |
| R | Bật/tắt auto-attack |
| Tab | Chuyển mục tiêu |
| Space | Nhảy |
| F | Tương tác NPC |
| I | Mở hành trang |
| C | Thông tin nhân vật |
| ESC | Đóng cửa sổ / Hủy chọn |

## Môn phái

| Môn Phái | Loại | Đặc điểm |
|----------|------|----------|
| Thiếu Lâm | Tank / Cận Chiến | HP cao, phòng thủ mạnh |
| Cái Bang | DPS / Cận Chiến | Sát thương cao, nhanh |
| Đường Môn | DPS / Tầm Xa | Ám khí, độc dược |
| Thiên Vương | Tank / Hỗ Trợ | HP cực cao, buff đồng đội |
| Ngũ Độc | Pháp Sư / Debuff | MP cao, kỹ năng diện rộng |
| Võ Đang | Pháp Sư / Cận Chiến | Cân bằng, hồi phục |
| Đào Hoa | Hỗ Trợ / Debuff | Tốc độ cao, heal |
| Côn Lôn | DPS / Cận Chiến | Kiếm pháp mạnh mẽ |

## Khu vực

- **Tân Thủ Thôn** - Làng tân thủ (Level 1-5)
- **Lâm An Thành** - Thành phố, NPC mua bán
- **Hoàng Sơn** - Rừng núi, quái vật (Level 3-15)
- **Giang Hồ Lộ** - Đường hiểm, Boss (Level 10-25)
- **Tử Cấm Thành** - Khu vực cao cấp (Level 20-40)
- **Mộ Dung Trang** - Trang viện (Level 8-22)
